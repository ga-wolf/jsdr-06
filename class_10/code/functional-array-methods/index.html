Functional Array Methods

Each one of the JS files focusses on one array method

In each one of them, there are 3 versions (all doing the exact same thing):

  • Our own version
  • Another version of our own (using a function)
  • A version where we use the array method itself

Remember that these are utilities! We don't need them - they can just make our code more performant and a little easier to reason about
